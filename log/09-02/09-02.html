I finally got past my resizable window dilemma after a while.

This is an attempt of explaining how this was done

    • Bunch of divs are created to take place as the borders of the resizable window and receive the mouse events
      When any of them are pressed, the function resizeWindow() is called
    • Bunch of variables are created to store useful informations about the Window and the events
    • A variable called 'tackledBorder' gets the ID of the border clicked and activates the resizing Event
    • As the mouse moves, the event happens and calculation based on the offset distance (top and left) and the height/width happens!
      • Each corner and side has their own different calculation

    • I also implemented a behavior to show the window open over the footer bar.
